# Xây dựng môi trường cho VPS

## 1. Đối với VPS mới khởi tạo

Chạy lệnh sau để setup môi trường web service:

```shell
cd nevent-config
bash config.sh
```

## 2. Đối với VPS đã thuê trước đó muốn chạy pipeline CI - CD

Chạy lệnh sau:

```shell
cd nevent-config
bash deploy.sh
```
